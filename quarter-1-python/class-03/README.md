## ðŸ’» Code Examples

### Basic if Statement
```python
age = 18
if age >= 18:
    print("You are an adult")
```

### if-else Statement
```python
temperature = 25
if temperature > 30:
    print("It's hot outside!")
else:
    print("It's not too hot")
```

### if-elif-else Statement
```python
score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")
```

### User Input
```python
name = input("What is your name? ")
age = input("How old are you? ")

```
